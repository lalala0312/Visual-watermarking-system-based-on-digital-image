# Visual-watermarking-system-based-on-digital-image
数字图像可视化水印系统的设计与实现

(LSB算法的实现)

---





### 程序介绍

* 使用Python实现了一个数字图形可视化水印系统，但我只需要其中的LSB功能，所以对原作责的其他内容进行了删减。
* main.py的GUI使用的Tkinter
* Main2.py增加了DES加密
* Main3.py增加了批量图片加密的部分

### 功能

* LSB基本算法
* DES基本加密算法



### 环境

* Python3.7

直接运行即可

```
python main.py
```
